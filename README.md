# Low Power VLSI Design using Cadence Virtuoso (FINFET Technology)

## Project Overview

This project focuses on the design and implementation of low-power digital circuits using **Transmission Gate Logic (TGL) implemented with FINFET technology**. The circuits were designed and simulated using **Cadence Virtuoso VLSI design environment**.

The work demonstrates how FINFET-based transmission gate circuits can significantly reduce power consumption, transistor count, and propagation delay compared to conventional CMOS logic.

## Design Methodology

1. Digital circuits were designed using **Transmission Gate Logic structures**.
2. FINFET devices were used instead of traditional planar CMOS to improve electrostatic control and reduce leakage current.
3. The circuits were implemented in **Cadence Virtuoso schematic and layout environment**.
4. Transient simulations were performed to verify circuit functionality.
5. Performance parameters such as:

   * Power consumption
   * Propagation delay
   * Power-Delay Product (PDP)
   * Transistor count
     were analyzed.

## Key Results

Simulation results showed that the proposed **TGL-FINFET circuits** achieve:

* Lower power consumption
* Reduced propagation delay
* Improved power-delay product
* Reduced transistor count
* Better energy efficiency

These improvements make the circuits suitable for **low-power VLSI applications such as memory systems, communication circuits, and digital signal processing architectures**.
